<h1 class="title">Яндекс (1 задание, Стажировка осень 2024, Контест)</h1>
<p><b>Время: 1 сек.<br>Память: 256 Мб<br>Сложность: ?</b></p>
<p>А. Кроссворд</p>
<p>Решение кроссвордов - популярное времяпрепровождение. Новый сервис Яндекс.Кроссоворды будет предлагать пользователям
подобрать кроссворд, распечатать его и решить на бумаге. Однако? в таком случае проверить правильность решения кроссворда
непросто.
Кроссворд расположен на клетчатом поле, состоящим из R строк и С столбцов. Каждая клетка покрашена в белый цвет (туда можно
вписать букву) или в черный. Пользователь сервиса будет решать кроссворд и выписывать слова в белые клетки горизонтально (слева-
направо) или вертикально (сверху-вниз) по одной букве в клетке. После решения кроссворда незаполненных белых клеток не остается.
Словом называется горизонтальная или вертикальная последовательность клеток с буквами, ограниченная по краям черными клетками
или границами поля.</p>
<p>Для проверки правильности решения кроссворда пользователь должен ввести в качестве ответа лексикографически минимальное
слово (первое в алавитном порядке), причем длина слова должна быть не меньше 2 символов.</p>
<p>Ваша задача состоит в том, чтобы определить ответ по разгаданному кроссворду.</p>

<h2>Формат ввода</h2>

<p>В первой строке вводится два целых числа R и C (1 ≤ R, C ≤ 20) - количество строк и столбцов соответственно.
В следующих R строках вводится по одному слову, каждое слово состоит из прописных латинских букв и символов "#", обозначающих
черную клетку.</p>
<p>Гарантируется, что существует хотя бы одно слово длиной в 2 или более символов.</p>

<h2>Формат вывода</h2>
<p>Выведите лексикографически минимальное слово, длина которого не меньше 2 символов.</p>

<h2>Примечания</h2>
<p>В первом примере в кроссворде всего 3 слова: yd, yan и neх. Минимальным является nех
Во втором примере по вертикали есть 2 слова: cont и net. По горизонтали 3 слова: on, te, st. 
Обратите внимание, что слово "с" в первой горизонтали не подходит - оно состоит только из одной буквы.</p>
<p>В третьем примере все слова по горизонталям и вертикалям состоят из 2 или более букв, минимальным является еі.</p>
